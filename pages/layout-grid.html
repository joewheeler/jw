Bootstrap Grid

Bootstrap is "Mobile first" -- a responsive Web design practice that prioritizes consideration of smart phones and mobile devices when creating Web pages. Bootstrap 3 allows Web page elements (text, images, buttons, paragraphs, etc..) to scale accordingly as the viewing area, size and resolution changes from wide-screen desktop to tablet to small screen smartphone. Mobile-first elimates to need to make separate desktop and mobile versions of your Web site.

The powerful Bootstrap 3 grid comes in 4 sizes (or "breakpoints"). Tiny (for smartphones .col-xs-*), Small (for tablets .col-sm-*), Medium (for laptops .col-md-*) and Large (for laptops/desktops .col-lg-*). The 4 grid sizes enable you to control grid behavior on different devices (desktop, laptops, tablet, smartphone, etc..).
